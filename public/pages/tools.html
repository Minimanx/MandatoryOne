Tools

Powershell

Powershell is a Command-line Interface (CLI), used to perform tasks using written commands.
It is capable of navigating through your directories, deleting, copying, creating and moving files and much much more.
We also use other CLIs, such as npm and git, through Powershell.

Commands

cd - Used to change directory.
ls - Used to list items in the current directory.
mkdir - Used to make a new directory.
ni - Used to make a new item.
rm - Used to remove an item.
cp - Used to copy an item.
mv - Used to move an item.
pwd - Used to print the current working directory.

Nodemon

Nodemon is a tool for Node.js, that you can run your project with to have it automatically restart when changes have been made.
This is especially, and only, useful for development purposes.
Nodemon has a config file called nodemon.json, where you can change which programming languages it will monitor to register changes.

Commands

nodemon.cmd "filename.js" - Used to execute a file with Nodemon.

Git + GitHub

Git is a free and open source distributed version control system.
Git tracks changes to files and keeps track of your code history, saves repositories locally and much more.
Github is a cloud-based hosting service that you can use independently or manage through Git.
Github allows you to save and manage repositories in the cloud on an account and collaborate with others.

Commands

git clone "url here" - Used to clone an existing repo to the current directory.
git init - Used to initialize a new repo in the current directory.
git add -A - Used to tell Git to include all changes in the next commit.
git commit -m "your message here" - Used to "capture a screenshot" of the projects current state.
git push - Used to push your commit to the desired branch.

JSON

JSON stands for JavaScript Object Notation. It's a data format used to store and transfer data, conisting of arrays and key:value pairs.
As the name may suggest, the format is derived from JavaScribt objects.
It is especially useful since it is "global" and almost all programming languages have some way of parsing JSON.
