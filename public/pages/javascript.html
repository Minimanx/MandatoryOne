JavaScript

About

JavaScript is a programming language mainly used for client side execution in web pages.
However, it is also used in non-browser environments such as Node.js.

Variable

Variables are containers that we use for storing data.
There are 3 ways to declare variables.

  • var

  • let

  • const

Var should avoid being used. It has a global scope and is the least specific of the 3.
Const should be used when possible. It cannot be reassigned to. If a variable needs to be reassigned at some point, use let.
Let can be reassigned to and is therefore not immutable as some would say const is.
Both const and let are block scoped and only function within their given scope.

Objects

Objects in JavaScript are variables that can contain several values as properties / {key:"value"} pairs.

Type Coercion

Type coercion in JavaScript is something that can happen in JavaScript when comparing different values.
Generally speaking, the == operator coerces. That means 1 == "1" would result in true, even though the data type is different.
To avoid this we add an extra equals sign i.e. ===.
This, of course, also works for inequality. != vs !==.

Callback Functions

A callback function is a function that is passed to another function as a parameter.
This can be used to run functions parallel with other functions (asynchronous).

Loops

There are a few different ways to iterate through arrays in JavaScript.
Generally speaking, we don't want to use for loops or while loops to do so. For loops should be used for "finger counting" and while loops for when something needs to be executed until the intended job is finished.
The main built in functions for iterating through an array are:

  • forEach
  • map
  • filter

All these functions takes a callback function as an argument.
forEach executes the provided function on each element, but doesn't return anything (undefined).
map creates a new array, adds the result of calling the provided function on each element and then returns the new array.
filter similarly creates a new array, adds all elements that pass a test provided by the callback function and returns the new array.
