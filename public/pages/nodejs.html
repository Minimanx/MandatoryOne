Node.js

About

Node.js is a JavaScript runtime environment built on Chrome's JavaScript engine.
It is 100% serverside and designed to build scalable network applications.
Node.js uses the Node Package Manager (NPM) CLI to install and manage packages for your apps, among other things.

Node.js uses a file called package.json, located in the root folder of the application, as its build tool. It does 3 things:

  • Defines dependencies (packages/modules).

  • Defines the build steps (called "Scripts" in NPM).

  • Contains the meta data of the project.

Importing

Commands

node "filename.js" - Used to execute a file with node.
npm i "modulename" - Used to install a module to your project.
npm i - Used to install all necessary dependencies of the current project.
npm init - Used to setup your package.json file.
npm run "scriptname" - Used to run a script (build steps).
